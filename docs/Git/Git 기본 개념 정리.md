## Git 과 Subversion 의 차이

- 가장 큰 차이는 데이터를 다루는 방식인데, Git의 경우는 버전별(시간별)로 Snapshot을 찍는 개념이라는 점.
- SVN과 달리 Local Repository 가 존재하여 Repo에 파일을 추가 (add) 하고, 커밋(commit)하여 인터넷이 끊기더라도 로컬에 저장된 데이터로 작업을 계속할 수 있다는 장점.

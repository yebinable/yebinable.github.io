# Sitemap 정리

## 문제
- `include: [_posts]` 설정으로 인해 이미지 폴더가 URL로 sitemap에 추가됨
- 예: `https://yebinable.github.io/_posts/2025-11-20-folder/`

## 해결
- `include` 제거
- `keep_files`로 빌드 시 이미지만 보존
- sitemap은 자동으로 포스트 URL만 포함

## 확인
sitemap.xml에서 다음만 표시되어야 함:
- `/git-submodule/` (포스트)
- `/Github-Pages-노출하기/` (포스트)
- `/` (홈)
- `/archive/` (페이지)
- `/tags/` (페이지)

`_posts/폴더명/` URL은 제거됨

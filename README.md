# Yebin's Tech Log

기술과 성장의 기록을 담는 개발 블로그입니다.

## 🚀 빠른 시작

### 로컬에서 실행하기

```bash
# 의존성 설치
bundle install

# 로컬 서버 실행
bundle exec jekyll serve

# 브라우저에서 확인
# http://localhost:4000
```

## 📝 포스팅하기

**파일 위치:** `_posts/`  
**파일명 형식:** `YYYY-MM-DD-제목.md`

**예시:**
```markdown
---
title: "포스트 제목"
date: 2024-01-15
tags: [태그1, 태그2]
description: "포스트 설명"
---

포스트 내용을 여기에 작성합니다.
```

📖 **자세한 사용법은 [POSTING_GUIDE.md](POSTING_GUIDE.md)를 확인하세요!**

## 🎨 디자인

Gerous Jekyll 테마를 기반으로 한 미니멀하고 모던한 디자인입니다.

**주요 특징:**
- 깔끔한 타이포그래피
- 반응형 디자인
- 빠른 로딩 속도
- 읽기 편한 레이아웃

## 📂 구조

```
.
├── _posts/          # 블로그 포스트
├── _layouts/        # 레이아웃 템플릿
├── assets/          # CSS, JS, 이미지
├── images/          # 이미지 파일
├── index.html       # 홈페이지
├── archive.html     # 아카이브
├── tags.html        # 태그 페이지
└── about.html       # 소개 페이지
```
